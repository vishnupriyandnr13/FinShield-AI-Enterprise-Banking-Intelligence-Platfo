# FinShield-AI-Enterprise-Banking-Intelligence-Platfo
An AI-powered enterprise banking platform inspired by peak shopping events such as Black Friday 2025, featuring real-time fraud detection, dynamic risk scoring, RBI-compliant RAG assistance, bank statement analysis, and high-volume payment traffic simulation through an intuitive and modern user interface.

## Run and deploy

This contains everything you need to run your app locally.

View app: https://ai.studio/apps/9f5ab8bd-be20-4d0a-9e21-81d8fbeb7e5d

### Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
